# StudentIQ (StdIQ)

StudentIQ is an AI-powered learning companion designed to help students stay focused on their curriculum, strengthen subject mastery, and prepare effectively for competitive and national-level exams.

## Features

- **Personalized Mock Tests**: Difficulty-adaptive mock tests that adjust to your learning level
- **Daily Topic Guidance**: Personalized recommendations on topics to study
- **Smart Revision Workflows**: AI-powered revision schedules that optimize retention
- **Focus on Curriculum**: Stay aligned with your academic goals
- **Progress Tracking**: Monitor your performance with detailed analytics
- **Competitive Exam Prep**: Specialized preparation materials for national and competitive level examinations

## How It Works

1. **Set Your Goals**: Tell us about your curriculum, target exams, and learning objectives
2. **AI Assessment**: Our AI evaluates your current knowledge level and creates a personalized learning path
3. **Daily Practice**: Engage with adaptive mock tests and receive daily topic guidance tailored to you
4. **Smart Revision**: Follow AI-optimized revision schedules to maximize retention and exam readiness

## Local Development

To run the website locally:

```bash
# Clone the repository
git clone https://github.com/stdiq/stdiq.github.io.git

# Navigate to the directory
cd stdiq.github.io

# Start a local server (Python 3)
python3 -m http.server 8000

# Or using Node.js
npx http-server -p 8000
```

Then open your browser and navigate to `http://localhost:8000`
